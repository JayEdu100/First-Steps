Types of Selectors

  • Element Selectors - Select elements by their tag name.
    p { font-size: 16px; }
  • Class Selectors - Select elements by their class attribute.
    .my-class { background-color: yellow; }
  • ID Selectors - Select a single element by its ID attribute.
    #my-id { border: 1px solid black; }
  • Attribute Selectors - Select elements by their attributes.
    [href] { color: red; }
  • Pseudo-class Selectors - Select elements based on their state or position.
    a:hover { color: green; }
  • Pseudo-element Selectors - Select elements based on their position in the document.
    p::first-line { font-weight: bold; }

Combinators

  • Selectors - Select all elements on the page.
    * { margin: 0; padding: 0; }
  • Group Selectors - Select multiple elements with the same styles.
    h1, h2, h3 { color: blue; }
  • Chaining Selectors - Combine multiple selectors to target specific elements.
    div.my-class { background-color: lightgray; }
  • Descendant Selectors - Select elements that are descendants of a specified element.
    div p { color: gray; }
  • Child Selectors - Select elements that are direct children of a specified element.
    ul > li { list-style-type: square; }
  • Adjacent Sibling Selectors - Select elements that are immediately adjacent to a specified element.
    h2 + p { margin-top: 0; }
  • General Sibling Selectors - Select elements that are siblings of a specified element.
    h2 ~ p { color: purple; }
  • Attribute Value Selectors - Select elements based on specific attribute values.
    input[type="text"] { border: 1px solid blue; }

Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree and the layout of these boxes. It consists of four components: content, padding, border, and margin.

  • Content - The innermost part of the box where text and images appear.
  • Padding - The space between the content and the border. It can be used to create space around the content.
  • Border - The line that surrounds the padding and content. It can be styled with different widths, colors, and patterns.
  • Margin - The outermost part of the box that creates space between the element and other elements on the page.

Understanding the box model is crucial for designing layouts and controlling the spacing of elements on a webpage.

border-box

When using the border-box model, the width and height of an element include the content, padding, and border. This means that if you set an element's width to 200px and it has 20px of padding and a 2px border, the total width of the element will still be 200px. The content area will be reduced to accommodate the padding and border.
